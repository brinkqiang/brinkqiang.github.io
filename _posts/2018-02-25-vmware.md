# 1. 通过vmdk安装centos虚拟机
```
https://www.osboxes.org
下载对应版本的vmdk文件
```
```
1) 首先要打开VMware软件，点击创建一个新的虚拟机。
2) 点击下一步到 安装来源，选择稍后安装操作系统。然后点击下一步。
3) 客户机操作系统选择Linux，版本选择centos，点击下一步。
4) 网络连接使用网络地址转换。点击下一步。
5) 磁盘使用现有虚拟磁盘，点击下一步。
6) 点击浏览选择一个现有虚拟磁盘，点击打开，点击下一步，点击完成即可。
7) VMware中通过VMdk文件创建虚拟机成功。
```
