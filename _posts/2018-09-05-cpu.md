- CPU 操作的周期消耗一览

| 操作类型                     | CPU 时间片（周期数）      |
| ------------------------ | ----------------- |
| 简单寄存器操作（ADD, OR, etc.）   | <1                |
| 内存写入                     | \~1               |
| 延迟规避：整数与浮点之间切换           | 0-3               |
| if 判断正确分支                | 1-2               |
| 浮点/向量加法                  | 1-3               |
| 乘法（整数/浮点/向量）             | 1-7               |
| 返回错误与检查                  | 1-7               |
| L1 缓存读取                  | 3-4               |
| TLB Miss                 | 7-21              |
| L2 缓存读取                  | 10-12             |
| if 判断错误分支（预测错误）          | 10-20             |
| 浮点除法                     | 10-40             |
| 128-bit 向量除法             | 10-70             |
| 原子操作/CAS                 | 15-30             |
| C 语言函数直接调用               | 15-30             |
| 整数除法                     | 15-40             |
| C 语言函数间接调用               | 20-50             |
| C++ 虚函数调用                | 30-60             |
| L3 缓存读取                  | 30-70             |
| 主内存读取                    | 100-150           |
| NUMA：不同 Socket 的原子操作/CAS | 100-300（估算值）      |
| NUMA：不同 Socket 的 L3 读取   | 100-300           |
| 分配与回收（小对象）               | 200-500           |
| NUMA：不同 Socket 的主内存读取    | 300-500           |
| 内核调用                     | 1000-1500         |
| 线程上下文切换（直接成本）            | 2000              |
| C++ 异常抛出与捕获              | 5000-10000        |
| 线程上下文切换（总成本，包括缓存失效）      | 10000 - 1,000,000 |
